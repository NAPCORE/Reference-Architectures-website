KPI 1.21 is regarding “off-line” metadata information about data changes (update frequency and/or expected regular times of data updates, each as “Value +1”). Higher evaluation (next “Value +1”) is applied if Mobility DCAT-AP is used in such metadata information.<br/><br/><i>Note: Such information can be a part of metadata provided by the NAP even if that data is external.</i><br/><br/>Values:<br/><b>N/A:</b> NAP is a data platform with no provision of data<br/><b>0: </b>NAP provides data but no change/modification metadata about the data is provided<br/><b>Value+1</b> (for each of the following options):<br/>NAP provides change/modification metadata about the data:<br/><b>a. </b>NAP provides metadata about frequency of data changes<br/><b>b.</b> NAP provides metadata about typical dates/times of data change/modification<br/><b>c.</b> The metadata addressed in option “a” and/or “b” (above) is provided using Mobility DCAT-AP<br/><br/>Minimum acceptable value for interoperability: 1<br/>
