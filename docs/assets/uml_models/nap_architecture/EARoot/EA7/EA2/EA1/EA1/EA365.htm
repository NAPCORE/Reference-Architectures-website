The platform shall support the self-authentication of data providers (e.g., via e-mail) and at least one of the following authentication methods:<br/><b>a.</b> Authentication of data providers is double checked, i.e., by the data provider and by the NAP operator (e.g., via checking a state registry of legal bodies)<br/><b>b. </b>Usage of two-factor authentication and/or ID certificate (e.g., self-logging with secure national ID)<br/><br/><br/>Source: KPI 1.10 Security – providers verification<br/>
