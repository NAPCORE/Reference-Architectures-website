<b>Identification: </b>mdRS – mdMM; Metadata Requesting System - Metadata Management Module<br/><b>Type</b>: Programming Interface<br/><b>Description: </b>This interface is used for the retrieval of the metadata stored at NAP by automated system of Content Consumer. To allow for basic service provision and as a protection from denial of service attacks this interface may require authorization. The API shall provide similar search and retrieve functionality as web-based user-oriented UI. This interface shall support metadata exchange as specified by mobilityDCAT-AP.<br/><b>Related Documents</b>: <a href="file://https://napcore.eu/activity-wg2-interoperability-and-level-of-service-of-naps/"><font color="#467886"><u>https://napcore.eu/activity-wg2-interoperability-and-level-of-service-of-naps/</u></font></a><br/>
