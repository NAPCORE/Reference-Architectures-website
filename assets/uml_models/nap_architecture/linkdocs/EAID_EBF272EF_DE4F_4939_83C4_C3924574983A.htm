Communications Requirements

Data size and interval and maximum delay varies by the type of the data:

  • static data: small parking static data (500 kB/semester) to detailed road trajectory and attributes (2+ GB/semester)
  • semi-static data: road closures and traffic measures (500 kB/day)
  • dynamic data: incidents accidents, event data (500 kB/5 minutes)
  • status data: small area with parking sensors (200 kB/5 minutes) to travel time data for whole network (1 GB/5minutes)

Usually, use of this interface is expected for semi static, dynamic and status data, it could be to difficult to set up a machine for uploading static data twice a year.
