Management of the repository of declaration forms<br/>Requirements: All types of self-declaration forms (templates) with a detailed description and filling instructions are uploaded in the repository. Each template must be assigned with an unique ID. Full list of all self-declaration forms (templates) available on the web-portal (NAP or NB/CA) with the possibility to select a template, preview and/or download.<br/><br/>Reference to State diagram<br/>Add lifecycle:<br/>may withdraw the SD, changes need to be adopted. This can lead to a replacement of the old SD or a deletion of the existing SD.<br/><br/>Old SD should be replaced by updated SD's<br/>SD's stay valid until something makes the SD invalid.<br/>Invalid SD's can be stored in alignment to GDPR for statistical reasons, but should not be used for compliance assessment.<br/>
